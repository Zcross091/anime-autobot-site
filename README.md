# Anime Instagram Reels Bot 🤖

Automated Instagram bot specialized in posting **viral anime reels** every hour.

Built for anime content creators who want consistent growth with minimal effort.

---

## ✨ Features

- Automatic anime shorts downloading from YouTube
- AI-powered engaging captions + trending hashtags
- Posts every hour automatically
- Auto cleanup to save storage
- Session management to reduce ban risk

---

## 🚀 Quick Start

1. Clone this repo
2. Install `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and add your Instagram credentials
4. Run `python bot.py`
